# 노원구 · 서울특별시

> 노원구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-nowon.github.io/**

노원구는 불암산과 수락산이 병풍처럼 감싸 안은 서울 동북단의 자치구로, 갈대(蘆)가 무성했던 벌판(原)이라는 옛 지명을 그대로 이름에 새기고 있습니다. 상계·중계·하계·공릉·월계 다섯 개 법정동으로 이루어져 있으며, 1980년대 대규모 택지개발로 조성된 상계 신시가지가 구의 골격을 이룹니다. 서울과학기술대학교와 광운대학교, 서울여자대학교, 삼육대학교 등 여러 대학과 육군사관학교가 몰려 있어 교육과 학문의 색채가 짙은 고장이기도 합니다. 조선 명종의 어머니 문정왕후가 잠든 태릉이 자리해 왕실의 역사가 흐르는 한편, 폐선을 되살린 경춘선숲길과 화랑대 철도공원이 도심 속 산책 명소로 사랑받습니다. 아파트 단지와 산자락 녹지, 하천 산책로가 어우러져 정주형 주거도시의 편안함을 지닌 곳입니다.
